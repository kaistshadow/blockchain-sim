BLEEP Web Interface

You can execute the blockchain experiment by typing 'run' on command box.

The experiment will be automatically executed on back-end server.

Command list

  • run : run blockchain experiment. Following additional option is possible.

    • --nodenum "value" : specify the number of blockchain peer node. It should be larger than 2.

    e.g. run --nodenum 100

  • stop : stop the blockchain experiment.

Simulation Parameters

  • Node num : 500 (default)
  • Virtual simulation time : 150 seconds
  • Network module : asynchronous centralized proxy
